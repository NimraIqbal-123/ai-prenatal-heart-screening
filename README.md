An AI-powered web application designed to assist in the early detection of prenatal heart abnormalities using ultrasound images. The system combines Deep Learning, Computer Vision, and a modern web interface to help improve the screening process and support healthcare professionals.

## рџ“Њ Project Overview

Congenital Heart Disease (CHD) is one of the most common birth defects worldwide. Early detection during pregnancy can significantly improve treatment planning and patient outcomes.

This project uses Deep Learning models to analyze prenatal ultrasound images and classify them as Normal or Abnormal. The application provides an easy-to-use interface for image upload, prediction, and result visualization.
